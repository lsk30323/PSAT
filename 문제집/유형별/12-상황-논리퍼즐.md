# 상황판단 — 조건추론·논리퍼즐 학습지

> 국회직 8급 1차 PSAT 상황판단(20문항) 중 **시간 변동성이 가장 큰 변별 유형**.
> 난도 기준: **민경채 ~ 국가직 7급 사이**. 아래 문항은 7급·민경채 수준으로 재구성한 **창작 학습 문제**다.

---

## 개념 이해 (핵심 정리)

### 유형 정의
여러 **조건(제약)** 을 동시에 만족하는 **배치·순서·대응**을 찾는 문항. 하위 갈래:
- **자리 배치형**(좌석·층·방): "A는 B 바로 옆", "C는 끝자리 아님" → 칸 그림.
- **순서·일정형**(등수·발표 순서·요일): "갑은 을보다 먼저", "둘 사이 한 명" → 일렬 칸.
- **짝짓기·대응형**(사람↔직업/색/도시 매칭): "갑은 의사 아님" → **O/X 매트릭스**.
- **참·거짓(진실게임)형**: 진술 중 일부만 참/거짓 → 모순쌍으로 가정 제거.

### 핵심 스킬 (3단계 엔진)
1. **확정 조건부터 고정** — 위치가 거의 정해지는 조건("끝자리", "월요일", "유일한 ~", "3번")을 표·그림에 **먼저 박는다**.
2. **표/매트릭스로 경우의 수 가지치기** — 자리·순서는 칸 그림, 짝짓기는 O/X 격자. **머릿속 금지, 반드시 여백에 손으로.** 인접쌍은 `[BC]` 블록으로 묶어 슬라이딩.
3. **모순 제거** — 각 분기를 끝까지 밀어 충돌(O/X 부딪힘, 같은 칸 2명)이 나면 그 가지 폐기. 살아남는 가지가 답.

### 전략
- **강한 제약(=경우의 수가 가장 적은 조건) 먼저.** 약한 조건으로 시작하면 가지가 폭발한다. 선택지를 가장 많이 잘라내는 조건이 1순위.
- **선지 대입 역산.** "반드시 참/거짓", "가능한 것" 발문은 정공법보다 **선지를 직접 대입**해 조건 위반을 찾는 게 빠를 때가 많다.
- **O 하나 찍으면 즉시 그 행·열 전부 X 전파.** 다음 단독칸이 연쇄로 확정된다.
- **해가 복수여도 "발문이 묻는 칸"이 모든 분기에서 같으면 답이 나온다.** 전 배치를 확정하려 들지 말 것.
- **목표 2분 / 손절 3분.** 조건 7개↑·분기 3갈래↑·"모든 경우 열거" 발문은 후순위(△ 표시 후 패스).

### 대표 함정 3종
- **조건 일부 누락** — 단서 5~7개 중 하나를 빠뜨리면 오답 확정. **적용한 조건엔 체크 표시**하며 전수 소진.
- **인접 방향 오해** — "바로 옆 / 바로 다음 날 / 사이에 한 명"의 **방향·거리**를 반대로 읽음(좌↔우, 앞↔뒤, 위↔아래). 원형 좌석의 "오른쪽"은 **앉은 사람 기준(시계방향)**.
- **중복 배치 허용 여부 착각** — 한 자리에 한 명만인지, 한 항목이 여러 번 쓰일 수 있는지(예: 같은 색 중복) **전제부터 확정**.

## 실전 문제

### 문제 1 — 일렬 자리 배치 (5명, 1~5번 좌석)

워크온 헬스데이 행사에서 5명(가·나·다·라·마)이 1번부터 5번까지 한 줄로 앉는다. 좌석 번호는 왼쪽에서 오른쪽으로 1→5 순서다. 다음 조건이 모두 참일 때, **반드시 참인 것은?**

- ① 가는 양 끝(1번 또는 5번)에 앉는다.
- ② 나는 다 바로 오른쪽에 앉는다. (다-나 순서로 인접)
- ③ 라는 3번 자리에 앉는다.
- ④ 가는 라와 이웃하지 않는다.
- ⑤ 마는 나보다 왼쪽에 앉는다.

① 가는 5번 자리에 앉는다.
② 나는 다 바로 왼쪽에 앉는다.
③ 마는 4번 자리에 앉는다.
④ 다는 라보다 오른쪽에 앉는다.
⑤ 가와 다는 서로 이웃해 앉는다.

---

### 문제 2 — 순서·일정 배정 (5개 업무, 월~금)

워크온 개발팀이 5개 업무(가·나·다·라·마)를 월요일부터 금요일까지 하루에 하나씩 배정한다. 다음 조건이 모두 참일 때, **다음 중 옳은 것은?**

- ① 다는 월요일에 처리한다.
- ② 마는 금요일에 처리한다.
- ③ 나는 가 바로 다음 날에 처리한다. (가→나 연속)
- ④ 라는 화요일에 처리하지 않는다.
- ⑤ 가는 라보다 앞선 날에 처리한다.

① 가는 월요일에 처리한다.
② 나는 수요일에 처리한다.
③ 라는 화요일에 처리한다.
④ 마는 목요일에 처리한다.
⑤ 다는 가 바로 다음 날에 처리한다.

---

### 문제 3 — 짝짓기 매칭 (4명 ↔ 4직업)

갑·을·병·정 네 사람의 직업은 각각 의사, 교사, 개발자, 변호사 중 하나이며 서로 겹치지 않는다(1:1 대응). 다음 조건이 모두 참일 때, **반드시 참인 것은?**

- ① 갑은 의사가 아니다.
- ② 을은 개발자도 교사도 아니다.
- ③ 병은 의사도 개발자도 아니다.
- ④ 정은 의사가 아니다.
- ⑤ 변호사는 갑이 아니다.
- ⑥ 갑은 개발자가 아니다.

① 갑은 개발자다.
② 을은 변호사다.
③ 병은 교사다.
④ 정은 의사다.
⑤ 을은 의사이고, 정은 개발자다.

## 정답

`1. ④  2. ②  3. ⑤`

## 해설

### 문제 1 해설 — 끝자리·고정칸 먼저, 인접 블록 슬라이딩

**고정 조건부터:** ③ "라=3번"이 즉시 확정되는 가장 강한 단서다. 먼저 박는다.

```
[ _ | _ | 라 | _ | _ ]
   1   2   3    4   5
```

**강한 제약 = 인접 블록:** ② "나는 다 바로 오른쪽" → `[다][나]` 폭 2 블록(순서 고정). 라=3을 피해 들어갈 연속 칸은 **1-2** 또는 **4-5** 두 곳뿐. 두 경우로 가지치기:

- **경우 가: 다나=1-2** → `[다|나|라|_|_]`. 남은 4,5번에 가·마. ① 가는 양 끝 → 1,2가 찼으니 **가=5**, 마=4.
  - ⑤ 검사: 마(4) < 나(2)? → 4는 2보다 **오른쪽**이므로 "마가 나보다 왼쪽" **위반**. → **경우 가 폐기.**

- **경우 나: 다나=4-5** → `[_|_|라|다|나]`. 남은 1,2번에 가·마. ① 가는 양 끝 → 4,5가 찼으니 **가=1**, 마=2.
  - ④ 검사: 가(1)와 라(3) 이웃? 1과 3은 비인접 → "이웃하지 않는다" **충족**.
  - ⑤ 검사: 마(2) < 나(5) → 마가 나보다 왼쪽 **충족**.
  - → **경우 나 생존.**

**유일해:**

```
[ 가 | 마 | 라 | 다 | 나 ]
   1    2    3    4    5
```

**선지 판정:** ① 가=1번이므로 거짓. ② 나=5번, 다=4번 → 나는 다의 **오른쪽**이라 거짓. ③ 마=2번이므로 거짓. **④ 다(4)는 라(3)보다 오른쪽 → 참.** ⑤ 가=1, 다=4로 비인접 → 거짓. → **정답 ④.**

**함정 회피:** 인접쌍 ②를 "다나"가 아니라 "나다(나가 왼쪽)"로 거꾸로 읽으면 경우 가가 살아남아 복수해처럼 보인다. 블록의 **좌우 순서**를 단서 그대로 고정하고, ⑤의 방향("나보다 왼쪽")을 좌표로 확인하면 한 해로 닫힌다.

### 문제 2 해설 — 양 끝 고정 후 연속 블록 + 부정·선후 조건

**고정 조건부터:** ① 다=월, ② 마=금. 양 끝이 먼저 박힌다.

```
월=다 | 화=_ | 수=_ | 목=_ | 금=마
```

남은 **화·수·목**에 가·나·라를 배정한다.

**강한 제약 = 인접 블록:** ③ "나는 가 바로 다음 날"(가→나 연속). 화·수·목 안에서 가능한 연속쌍은 **(화,수)** 또는 **(수,목)**. 두 경우로 가지치기:

- **경우 가: 가나=(화,수)** → 라=목.
  - ④ 검사: 라=목 ≠ 화 → **충족**.
  - ⑤ 검사: 가(화) < 라(목) → 가가 앞 → **충족**.
  - → `월=다·화=가·수=나·목=라·금=마` **생존.**

- **경우 나: 가나=(수,목)** → 라=화.
  - ④ 검사: 라=화 → "라는 화요일 아님" **위반**. (⑤로도 위반: 가=수, 라=화 → 가가 라보다 뒤 → "가는 라보다 앞" 위반.)
  - → **경우 나 폐기.** (④·⑤ 두 단서가 동시에 잘라낸다.)

**유일해:**

```
월=다 | 화=가 | 수=나 | 목=라 | 금=마
```

**선지 판정:** ① 가=화요일 → 거짓. **② 나=수요일 → 참.** ③ 라=목요일(화요일은 가) → 거짓. ④ 마=금요일 → 거짓. ⑤ 다=월, 가=화 → 다는 가의 **바로 전날**(다음 날 아님) → 거짓. → **정답 ②.**

**함정 회피:** "바로 다음 날"(③)은 **직후 인접**이지 "그 뒤 어딘가"가 아니다. 가나를 떨어뜨려 배치하면 안 된다. 또 ④(부정)와 ⑤(선후)는 경우 나를 함께 잘라내므로, 한 조건만 적용하고 넘어가지 말고 **남은 조건에 모두 체크**해 전수 소진해야 한다.

### 문제 3 해설 — O/X 매트릭스: X 채우고, O 찍으면 즉시 전파

**격자 작성(행=사람, 열=직업):** 조건 ①~⑥의 부정을 X로 모두 기입한다.

| | 의사 | 교사 | 개발자 | 변호사 |
|---|---|---|---|---|
| 갑 | X(①) |  | X(⑥) | X(⑤) |
| 을 |  | X(②) | X(②) |  |
| 병 | X(③) |  | X(③) |  |
| 정 | X(④) |  |  |  |

**1단계 — 행에 X가 3개면 남은 칸 O (확정).** 갑 행은 의사X·개발자X·변호사X → 남은 **교사=O(갑)** 확정.

**2단계 — O 즉시 전파.** 갑=교사 → **교사 열**의 을·병·정에 X 전파.

| | 의사 | 교사 | 개발자 | 변호사 |
|---|---|---|---|---|
| 갑 | X | **O** | X | X |
| 을 |  | X | X |  |
| 병 | X | X(전파) | X |  |
| 정 | X | X(전파) |  |  |

**3단계 — 다시 단독칸 스캔.** 병 행은 의사X·교사X·개발자X → 남은 **변호사=O(병)** 확정. → **변호사 열** 을·정에 X 전파.

| | 의사 | 교사 | 개발자 | 변호사 |
|---|---|---|---|---|
| 갑 | X | O | X | X |
| 을 |  | X | X | X(전파) |
| 병 | X | X | X | **O** |
| 정 | X | X |  | X(전파) |

**4단계 — 연쇄 확정.** 을 행은 교사X·개발자X·변호사X → **의사=O(을)**. 정 행은 의사X·교사X·변호사X → **개발자=O(정)**.

**유일해:**

```
갑 = 교사,  을 = 의사,  병 = 변호사,  정 = 개발자
```

**선지 판정:** ① 갑=교사 → 거짓. ② 을=의사 → 거짓. ③ 병=변호사 → 거짓. ④ 정=개발자 → 거짓. **⑤ 을=의사 ∧ 정=개발자 → 둘 다 참 → 참.** → **정답 ⑤.**

**함정 회피:** 갑이 교사로 확정되는 이유는 ①·⑤·⑥ **세 부정 조건이 의사·변호사·개발자를 한꺼번에 막기 때문**이다(조건 누락 시 못 잡는다). 또 O를 하나 찍을 때마다 **그 행과 열 양방향으로 X를 전파**해야 다음 단독칸이 드러난다 — 전파를 빼먹으면 "을=의사"가 끝까지 안 보여 시간을 잃는다.
