# 자료해석 — 단순 수치 읽기·계산 유형 학습지

> 국회직 8급 PSAT 대비 / 「개념 이해 → 실전 문제 → 정답 → 해설」
> 영역: 자료해석(2교시, 20문항·45분) / 유형: 표·그래프 값 직접 읽기 + 단순 사칙연산으로 선지 진위 판단
> ※ 아래 문항의 수치·소재는 KOSIS·국민건강영양조사 스타일로 재구성한 **학습용 창작**(기출 전문 복제 아님).

---

## 개념 이해 (핵심 정리)

**유형 정의.** 표·그래프에서 필요한 값을 직접 읽고, 합·차·비중·증가율·1인당 같은 **단순 사칙연산**으로 선지의 ○/×를 가르는 유형. 발상이 어려운 게 아니라 "시간 잡아먹는 계산의 연속"이라는 점이 본질이다. 자료해석 점수의 **바닥을 까는 기본 득점 블록**(20문항 중 대략 7~10문항에 포함).

**핵심 스킬.**
- **분수비교 = 교차곱**: a/b vs c/d는 나눗셈 없이 `ad vs bc` 부등호만. (ad > bc면 a/b > c/d)
- **"○% 이상" = 경계값 1개**: 전체 × 0.○를 만들어 분자와 직접 비교(나눗셈 불필요).
- **증가율 = 변화량 / 기준값(전년값)**: 분모가 작을수록 증가율이 크게 튄다.
- **유효숫자 어림산**: 앞 2~3자리만 남겨 대소만 판정.

**풀이 전략(접근 순서).**
1. **제목·단위·각주 먼저(10~15초)** — 천/백만/%인지, "전년 대비·누적·X 제외" 조건이 있는지. 함정의 8할이 여기서 나온다.
2. **선지를 먼저 읽고 필요한 셀만 특정** — 표 전체를 계산하지 않는다.
3. **어림셈으로 명백한 선지부터 소거** → 정확 계산은 애매한 1~2개에만.
4. **ㄱㄴㄷ 보기형**은 선지에 **자주 등장하는 보기부터** 계산해 소거.

**문항당 목표시간.** 단순확인·단순계산 **90초~2분**. 약간 꼬인 것 2분 30초 한도.
**'버릴 문제' 기준.** 표·그래프 2개 결합 + 선지마다 다중 분수비교면 시간소비형 → 30초 훑고 과하면 보류. **정확 계산을 3회 돌려도 선지가 2개로 안 좁혀지면 즉시 보류 후 이동.**

**대표 함정.**
- **단위 혼동**: 천 명↔만 명, 천 원↔백만 원 (표마다 단위가 다를 수 있음).
- **각주 조건 누락**: "전년 대비/제외/반올림값".
- **비율 ↔ 실수(절대값) 혼동**: 항목 절대값이 늘어도 전체가 더 크면 비중은 감소.
- **증가율 ↔ 증가량 혼동**, **% ↔ %p 혼동**: 증가'량' 1위 ≠ 증가'율' 1위.

## 실전 문제

### 문제 1.
다음은 어느 만보기 앱의 「연도별·권역별 일 평균 걸음수 측정 참여자 수」이다. 이에 대한 설명으로 **옳지 않은** 것은?

(단위: 천 명)

| 연도 | 수도권 | 영남권 | 호남권 | 합계 |
|---|---|---|---|---|
| 2023 | 1,420 | 980 | 610 | 3,010 |
| 2024 | 1,560 | 1,030 | 640 | 3,230 |

① 2024년 3개 권역 합계는 전년 대비 220천 명 증가하였다.
② 2024년 수도권 참여자는 같은 해 영남권 참여자의 1.5배 이상이다.
③ 2023년과 2024년 모두 호남권 참여자 수가 세 권역 중 가장 적다.
④ 2023년 수도권 참여자 수는 142만 명이다.
⑤ 2024년 영남권 참여자는 2023년 영남권 참여자보다 60천 명 많다.

### 문제 3.
다음은 「공공기관 걷기 챌린지」에 참가한 3개 기관의 참가자 수와 총 적립 포인트이다. 이에 대한 설명으로 **옳은** 것은? (단, 1인당 평균 포인트 = 총 적립 포인트 / 참가자 수)

| 기관 | 참가자 수(명) | 총 적립 포인트(천 점) |
|---|---|---|
| A | 250 | 1,500 |
| B | 400 | 1,800 |
| C | 150 | 1,200 |

① 1인당 평균 적립 포인트는 A기관이 세 기관 중 가장 많다.
② C기관의 1인당 평균 적립 포인트는 8,000점이다.
③ B기관의 총 적립 포인트는 C기관 총 적립 포인트의 2배 이상이다.
④ 참가자 1명당 적립 포인트는 B기관이 A기관보다 많다.
⑤ 세 기관의 1인당 평균 적립 포인트를 모두 더하면 20,000점을 넘는다.

## 정답

`1. ⑤   2. ③   3. ②`

## 해설

### 문제 1 — 정답 ⑤
**단위 = 천 명**, 합계 열이 이미 주어졌다(재합산 불필요). 선지가 지목한 셀만 본다.

- **①** 합계 열만 빼면 끝. 3,230 − 3,010 = **220천 명 증가** → **옳음(○)**.
- **②** "1.5배 이상" = 경계값 1개. 영남권 1,030 × 1.5 = **1,545**. 수도권 1,560 ≥ 1,545 → **옳음(○)**.
- **③** 각 행에서 최솟값만 스캔: 2023년 610(호남) < 980 < 1,420, 2024년 640(호남) < 1,030 < 1,560 → 두 해 모두 호남권 최소 → **옳음(○)**.
- **④** **단위 환산 함정**. 1,420천 명 = 1,420 × 1,000 = 1,420,000명 = **142만 명** → **옳음(○)**.
- **⑤** 영남권 1,030 − 980 = **50천 명** 증가인데 선지는 "60천 명 많다"고 함 → **틀림(×)** → **정답**.

**함정 회피.** ⑤처럼 차이값을 어림하지 말고 끝자리까지 정확히 빼라(30·80 같은 깔끔한 수에서 10 단위 오차를 흘려보내기 쉽다). ④의 "천 명↔만 명" 환산은 1,000배이므로 0의 개수만 세면 즉시 판정된다.

### 문제 2 — 정답 ③ (ㄱ, ㄷ)
**비중(실천율) 대소는 나눗셈 대신 교차곱**으로 끝낸다.

- **ㄱ.** 312/720 (2022) vs 290/640 (2023)을 교차곱으로 비교.
  - 312 × 640 = **199,680**
  - 290 × 720 = **208,800**
  - 208,800 > 199,680 이므로 **290/640 > 312/720** → 2023년 실천율이 더 높다 → **옳음(○)**.
  - (검산용 어림: 312/720 ≈ 43.3%, 290/640 ≈ 45.3% → 일치.)
- **ㄴ.** "45% 이상" = 경계값 1개. 2022년 전체 720 × 0.45 = **324**. 실천자 312 < 324 → 45% 미만 → **틀림(×)**.
- **ㄷ.** 전체 조사 인원만 비교: 640(2023) < 720(2022) → **2023년이 적다** → **옳음(○)**.

옳은 보기는 **ㄱ, ㄷ** → 정답 **③**.

**함정 회피.** ㄱ에서 "실천자 수가 줄었으니(312→290) 실천율도 낮아졌다"고 단정하면 오답이다. 분모(전체)가 720→640으로 **더 크게 줄어** 비율은 오히려 올라간다 — **비율 ↔ 실수(절대값) 혼동**의 전형. ㄴ은 나눗셈으로 43.3%를 굳이 구하지 말고 `720×0.45 = 324`와 312를 직접 비교하면 한 번에 끝난다.

### 문제 3 — 정답 ②
**1인당 = 총 적립 포인트 / 참가자 수.** 단위가 "천 점"임에 주의(환산 함정).

먼저 1인당(천 점)을 구한다.
- A: 1,500 / 250 = **6.0천 점 = 6,000점**
- B: 1,800 / 400 = **4.5천 점 = 4,500점**
- C: 1,200 / 150 = **8.0천 점 = 8,000점**

- **①** 1인당 최대는 C(8,000) > A(6,000) → A가 가장 많다는 것은 **틀림(×)**.
- **②** C는 1,200천 점 ÷ 150명 = 8천 점 = **8,000점** → **옳음(○)** → **정답**.
- **③** "2배 이상" 경계값: C총 1,200 × 2 = 2,400. B총 1,800 < 2,400 → **틀림(×)**.
- **④** 1인당 B(4,500) < A(6,000) → "B가 A보다 많다"는 **틀림(×)**.
- **⑤** 1인당 합 = 6,000 + 4,500 + 8,000 = **18,500점** < 20,000 → **틀림(×)**.

옳은 것은 ② 하나뿐 → 정답 **②**.

**함정 회피.** ① ↔ ④에서 "총 포인트가 많은 기관(B=1,800)이 1인당도 많다"고 착각하기 쉽다. 분모(참가자 400명)가 커서 1인당은 오히려 가장 적다 — **총량 ↔ 1인당(평균) 혼동**. 또 "천 점" 단위를 놓치면 ②의 8,000점을 8점·8천점 사이에서 헷갈리므로, 표 단위 행을 먼저 ○표 치고 시작한다.
