# 상황판단 — 수리·계산·게임 유형 [학습지]

> 국회직 8급 PSAT 상황판단 대비 / 「개념 이해 → 실전 문제 → 정답 → 해설」
> 난도 기준: 민경채~국가직 7급 수준. 아래 문제는 **학습용으로 숫자·규칙을 창작**한 것이며 기출 전문 복제가 아니다.

---

## 개념 이해 (핵심 정리)

### 1. 유형 정의
상황판단의 **퀴즈형** 중 "조건을 **계산**하거나 **경우를 따지는**" 문제군. 다섯 갈래로 나뉜다.

| 갈래 | 무엇을 묻나 | 정식화 도구 |
|---|---|---|
| **최소비용 / 최대이익** | 제약(하한·과락) 아래 비용 최소화·이익 최대화 | 부등식 + 극단 배분 |
| **점수·임계 판정** | 합산 점수가 기준값 초과/이상이면 대상 지정 | 산식 + 경계 비교 |
| **경우의 수·대진(리그/토너먼트)** | 총 경기 수, 진출 보장 최소 승점 | 조합 공식 + 완전탐색 |
| **게임 최적 선택** | 유한한 시간·자원에서 최대 점수 | 효율 정렬(배낭) |
| **규칙·암호 따라가기** | 변환 규칙 적용 결과 | 절차 순차 적용 |

### 2. 핵심 스킬
- **정식화**: 문제를 읽자마자 **식 / 부등식 / 표** 중 무엇으로 풀지 즉시 결정.
  - 비용·점수 → 부등식, 경우 확정 → 표(격자), 대진 → 조합 공식.
- **극단(경계)부터**: 최댓/최솟값 문제는 모든 경우를 나열하지 말고 "가장 클 때 / 작을 때"로 범위를 좁힌다. 비용 최소화는 **싼 자원에 몰고 비싼 자원은 하한에 붙인다**.
- **대칭·규칙 발견**: 반복·게임형은 한두 라운드 손으로 돌려 **주기·단조성·불변량(합·차가 일정)**을 찾아 일반화. 끝까지 손계산하지 않는다.
- **선지 역이용**: 답이 수치·대상이면 선지의 극단값·중간값부터 역대입.

### 3. 전략 (완전탐색 폭발형 손절)
- **3분 룰**: 진입 후 3분 내 식·표가 안 잡히면 즉시 별표 후 다음으로. **어려운 퀴즈 1개(6~7분) = 쉬운 문제 3개.**
- 분기가 수십 가지로 갈라지고 일반화 단서가 안 보이면 **즉시 포기 후보**.
- 대진·배낭형은 "전부 나열" 대신 **공식·효율 정렬**로 계산량을 1/10로 줄인다.
- 규칙·암호형은 난도 하 → **반드시 득점**(1~1.5분 컷).

### 4. 대표 함정
- **경계 등호("이상/초과", "이하/미만")**: 합 200점 *이상* vs 점수 7점 *초과*. 등호 포함 여부로 답이 갈린다. → 풀기 전 밑줄.
- **개별 하한(과락) 누락**: "한 과목이라도 50점 미만이면 탈락" 같은 개별 제약을 합계만 보다 빠뜨림.
- **동점·우선순위·예외 규칙**: 게임 동점 처리, 진출 시 동률 규칙을 놓치면 정답이 한 칸 어긋난다.
- **단위·횟수 오산**: 풀리그 총 경기 = **n(n−1)/2**(n²로 과대계산 금지), 라운드 수·일수.
- **효율 함정(게임)**: "단위당 효율 1등"만 채우면 **시간 꼬리가 남아 손해** — 제한시간을 **딱 채우는 조합**이 더 높을 수 있다.
- **형식 제약**: 두 자리 수에서 `01` 불가(최솟값 10), 정수·음수 제약.

## 실전 문제

### 문제 1. (최소비용형)

만보기 앱 **워크온**의 사내 자격 인증은 **A·B·C 세 과목**(각 100점 만점, 점수는 0~100 사이 **정수**)으로 본다. 인증 합격 조건은 다음 두 가지를 **모두** 만족하는 것이다.

- (가) **세 과목 합계 200점 이상**
- (나) **모든 과목 각각 50점 이상** (한 과목이라도 50점 미만이면 불합격)

각 과목은 **1점을 받기 위해 필요한 수강료**가 다음과 같다.

| 과목 | A | B | C |
|---|---|---|---|
| 1점당 수강료(원) | 300 | 500 | 800 |

총 수강료(= 세 과목 각각 점수 × 1점당 수강료의 합)를 **가장 적게** 들이고 합격하려 할 때, 그 **최소 총 수강료**는?

① 90,000원 ② 95,000원 ③ 100,000원 ④ 105,000원 ⑤ 120,000원

### 문제 2. (대진·승점형)

워크온 사내 걷기 대회 결승 토너먼트 진출을 가리기 위해 **4개 팀(가·나·다·라)**이 **단판 풀리그(라운드 로빈, 모든 팀이 서로 한 번씩 경기)**를 치른다. 경기 결과 승점은 **승 3점, 무승부 1점, 패 0점**이며, 리그 종료 후 **승점 상위 2개 팀**이 결승에 진출한다.

- 승점이 같은 팀이 있어 상위 2자리를 가릴 수 없는 경우, 그 팀들의 진출은 **확정되지 않은 것**으로 본다(즉, 동점이 끼면 "진출 보장"이라 할 수 없다).

다음 중 옳은 것은?

① 리그의 총 경기 수는 8경기이다.
② 한 팀이 얻을 수 있는 최대 승점은 6점이다.
③ 어떤 팀이 승점 6점을 얻으면, 다른 경기 결과와 관계없이 결승 진출이 보장된다.
④ 어떤 팀이 승점 7점을 얻으면, 다른 경기 결과와 관계없이 결승 진출이 보장된다.
⑤ 모든 경기가 무승부로 끝나면 네 팀의 승점 합계는 6점이다.

### 문제 3. (게임 최적 선택형)

워크온 앱의 **60분 챌린지**에서 참가자는 아래 세 미션을 **원하는 만큼 반복**(각각 0회 이상 정수 횟수) 수행할 수 있다. 단, **누적 소요시간이 60분을 넘으면 안 된다**(초과 불가). 미션 1회당 소요시간과 획득 점수는 다음과 같다.

| 미션 | 1회 소요시간 | 1회 획득 점수 |
|---|---|---|
| A (등산로) | 25분 | 50점 |
| B (공원걷기) | 20분 | 38점 |
| C (실내자전거) | 15분 | 27점 |

60분 안에 얻을 수 있는 **최대 점수**는?

① 100점 ② 108점 ③ 114점 ④ 115점 ⑤ 120점

## 해설

### 문제 1 — 정답 ②

**정식화.** 점수를 (A, B, C), 제약은 **A+B+C ≥ 200** 그리고 **A ≥ 50, B ≥ 50, C ≥ 50** (모두 정수, ≤ 100). 목표는 비용 `300A + 500B + 800C` 최소화.

**극단 전략(싼 자원에 몰기).**
1. 세 과목의 개별 하한을 먼저 바닥에 깐다 → (50, 50, 50), 합계 = 150.
2. 합계 200을 채우려면 **추가로 50점**이 필요하다.
3. 이 50점을 **1점당 수강료가 가장 싼 과목 A(300원)**에 몰아준다 → A = 50 + 50 = **100** (상한 100 이내 OK), B = 50, C = 50.
4. 비용 = 100×300 + 50×500 + 50×800 = 30,000 + 25,000 + 40,000 = **95,000원** → **②**.

**오답 근거.**
- ① 90,000원: 더 싸게 만들 조합이 없다. (가장 싼 A에 100점을 다 줘도 나머지 B·C는 하한 50을 깔아야 하므로 95,000원이 바닥.)
- ③ 100,000원 / ④ 105,000원: (50,100,50) = 105,000원처럼 비싼 과목에 점수를 준 경우. 비용 최적 아님.
- ⑤ 120,000원: (50,50,100), 가장 비싼 C에 50점을 더 준 최악 배분.

**함정 회피.** ⑴ 과락(나) 무시하고 (200,0,0) 또는 (100,100,0)으로 두면 **C=0이 50 미만**이라 불합격 — 개별 하한을 반드시 깔 것. ⑵ "합 200만 보고 균등 (67,67,66)" 으로 나누면 106,400원으로 최적이 아니다. **싼 과목 몰빵 + 나머지는 하한**이 핵심.

*(검산: 50~100 전 범위 완전탐색 결과 최소 95,000원 = (100,50,50)으로 일치.)*

### 문제 2 — 정답 ④

**기본 수치.**
- 총 경기 수 = **n(n−1)/2 = 4·3/2 = 6경기** → ①(8경기) **오답**.
- 한 팀의 경기는 3경기. **3전승 = 9점**이 최대 → ②(6점) **오답**.
- 전부 무승부면 경기당 양 팀 1점씩 = 경기당 2점, 6경기 → 합계 **12점** → ⑤(6점) **오답**.

**핵심: 진출 보장 최소 승점.** "보장"은 **나머지 경기 결과가 어떻게 나오든** 2위 이내가 확정된다는 뜻. 동점이 끼면 보장이 깨진다(문제 단서).

**③ 승점 6점 — 보장 안 됨(반례 존재).**
6경기가 모두 승부(무승부 0)로 끝나면 총 6승이 분배된다. 예컨대 **가·나·다가 각각 2승(=6점), 라가 0승(=0점)** 이 되는 결과가 가능하다. 이때 승점은 (가, 나, 다, 라) = **(6, 6, 6, 0)**. 6점인 팀이 셋이라 상위 2자리를 셋이 다투므로 **진출 불확정** → 승점 6점은 보장이 아니다. → ③ **오답**.

**④ 승점 7점 — 보장됨.**
승점 7점 = 2승 1무. 만약 나 외에 **7점 이상인 팀이 2팀 더** 있으려면 세 팀이 각각 최소 7점 = 합 21점 이상을 가져야 하는데, 4팀 총 승점은 (무승부 수에 따라) 최대 18점(6경기 전부 승부 시 6승×3=18)에서 12점(전부 무승부) 사이라 **21점은 불가능**하다. 따라서 7점이면 **나보다 승점이 같거나 높은 다른 팀이 최대 1팀** → 항상 2위 이내 확정. → ④ **정답**.

**함정 회피.** ⑴ 총 경기를 n²=16이나 n(n−1)=12로 과대계산하지 말 것(단판 풀리그는 n(n−1)/2). ⑵ "6점이면 2승이니 충분하겠지"가 함정 — **동점 3팀** 시나리오 때문에 보장선은 한 칸 위인 **7점**이다. 동점 처리 규칙을 반드시 확인.

*(검산: 6경기 모든 결과(3⁶=729가지) 완전탐색. 팀 승점 6점일 때 6점 이상 타팀이 2팀인 분포 (6,6,6,0)·(6,6,0,6) 등 존재 → 6 불충분. 7점일 때 7점 이상 타팀 최대 1팀 → 7 충분. 진출 보장 최소 승점 = 7로 확인.)*

### 문제 3 — 정답 ④

**정식화(배낭).** 미션 A·B·C를 각각 a·b·c회(0 이상 정수) 했을 때
- 시간 제약: **25a + 20b + 15c ≤ 60**
- 최대화 대상: 점수 **50a + 38b + 27c**

**효율 함정 점검.** 1분당 점수 효율은 A = 50/25 = **2.00**(1등), B = 38/20 = 1.90, C = 27/15 = 1.80. 효율만 보고 **A에 몰면**: A 2회 = 50분에 100점, **남은 10분으로는 어떤 미션도 못 해** → 100점에서 멈춤(①의 함정).

**제한시간을 딱 채우는 조합 탐색.** 60분을 남김없이 쓰는 조합을 따져본다.
- B 3회: 20×3 = 60분 → 38×3 = **114점** (③)
- C 4회: 15×4 = 60분 → 27×4 = 108점 (②)
- **A 1 + B 1 + C 1회: 25+20+15 = 60분 → 50+38+27 = 115점** ← 최대
- A 1 + C 2회 + (잔여 5분 못 씀): 55분에 104점

→ **A·B·C를 한 번씩** 하여 60분을 정확히 채울 때 **115점**으로 최대 → **④**.

**오답 근거.**
- ① 100점: 효율 1등 A만 2회 채운 함정(시간 10분 낭비).
- ② 108점 / ③ 114점: 한 미션만 반복(C 4회 / B 3회). 60분은 채우나 **혼합이 더 높다**.
- ⑤ 120점: 60분 안에 만들 수 없는 점수(어떤 조합도 115점이 상한).

**함정 회피.** 게임 최적화는 "효율 1등 몰빵"이 항상 정답이 아니다. **제한시간(60분)을 빈틈없이 채우는 조합**을 함께 따져야 한다. 효율 높은 행동이 시간 단위가 커서 **꼬리 시간을 못 쓰면 손해**라는 점이 핵심.

*(검산: a≤3, b≤4, c≤5 전 범위 완전탐색 결과 시간≤60 제약하 최대 = 115점 = (a,b,c)=(1,1,1)로 일치.)*

## 한 장 요약(체득용)
- 정식화는 **식 / 부등식 / 표** 중 하나로 즉시 분류.
- 비용 최소화 = **싼 자원 몰빵 + 나머지 하한**. 점수 최대화(게임) = **제한시간 딱 채우는 조합**(효율 1등 몰빵 함정 주의).
- 대진은 **n(n−1)/2**. 진출 보장선은 **동점 시나리오** 때문에 한 칸 올라간다(6 → 7).
- 경계 등호("이상/초과")·과락·동점은 **풀기 전 밑줄**.
- **3분 안 풀리면 별표 후 이동.** 어려운 퀴즈 1개 = 쉬운 문제 3개.
