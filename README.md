# 국회직 8급 PSAT 합격 패키지 (생성 산출물)

PSAT 유형 **15개를 1:1 전문 에이전트로 병렬 조사**하고 **검수관**이 합격 루브릭으로
채점·통합한 멀티 에이전트 오케스트레이션의 산출물이다. 대상 시험은 **국회직 8급 1차 PSAT**
— 언어논리 20 + 상황판단 20(1교시 90분 통합) + 자료해석 20(2교시 45분) = **60문항**,
경쟁 합격선 2025년 약 80점/48·60 (국회사무처공고 제2025-22호).

## 검수 3단계 (중요)

| 단계 | 기준 | 결과 | 파일 |
|---|---|---|---|
| 1 | **4축** (비중 근거 / 전략 / 예제카드 ≥8 / 함정) | **15/15 합격** | `판정표.md` |
| 2 | **5축 강화** (+ ①회차 실측 비중 ③실수치 worked example ⑤1차 출처) | **15/15 보완** | `판정표-엄격.md` |
| 3 | 위 지적대로 **보강 후 5축 재채점** | **15/15 합격 전환** | `판정표-엄격-2차.md` |

5축 강화 기준에서 처음엔 전 유형이 보완으로 강등됐다(결정타: ③ 카드에 실제 계산이 없음, ⑤ 출처가
위키·블로그 위주). 이후 **각 섹션에 worked example(실수치 계산·판정)과 1차 출처를 보강**해
다시 채점한 결과 **15개 전부 합격**으로 전환됐다. 보강 과정에서 08(표44/그래프40% 수치를 1차
분석으로 추적)·11(법조문 비중 56~60%→48~52% 하향 정정) 등 **자기교정**도 이뤄졌다.

## 생성·보강 방식 / 한계 (투명성)

- Claude 멀티 에이전트(유형별 전문가 15 + 검수관)로 생성·보강. 각 에이전트는 **WebSearch**로
  한국어 기출·1차 자료(공기출 기출원문, 사이버국가고시센터 gosi.kr, 국회채용 gosi.assembly.go.kr,
  공공데이터포털, 법제처 law.go.kr, 국회사무처공고)를 교차검증해 작성했다.
- **WebFetch가 다수 한국 사이트에서 403 차단**되어, 본문 정독이 아니라 **검색 스니펫 기반 합성**이다.
  1차 출처는 URL을 원문 위치로 수록했으되 일부는 직접 파싱이 아닌 스니펫 교차확인이다.
- **출제 비중은 추정치**다. 국회직 8급 자체 기출은 2025년 1회뿐이라, 모(母)시험인
  **국가직 7급·민경채(영역당 25문항)** 분포를 **20문항으로 환산**했다. 각 섹션에 추정 근거·라벨을 명시.
- **예제 카드의 worked example은 기출 전문을 복제하지 않고** 실수치 계산·판정 과정을
  '학습용 재구성'으로 재서술한 것이다(일부는 공개된 기출 회차·문항번호에 연결).
- 실제 시험 직전에는 **공개 기출 원문 대조**를 함께 권장한다.

## 사용법

1. `시간배분표.md`로 교시별 시간 설계(풀이 우선순위·버릴 순위)를 익힌다.
2. `PSAT합격-패키지.md` 2부에서 유형을 하나씩 골라 전략 → worked example로 구조를 훈련한다.
3. `판정표-엄격-2차.md`의 유형별 결과·근거를 학습 체크리스트로 활용한다.
