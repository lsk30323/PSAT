# PSAT 합격 검수 판정표 (국회직 8급)

> 검수관: 국회직 8급 PSAT 합격 검수관. 15개 유형 섹션을 PSAT 합격 루브릭(4축)으로 독립 채점.
> 합격 기준: 안정권 = 평균 80%(약 48/60), 세 영역(언어·자료·상황) 균형. 자료해석은 시간압박 최악 → 전략 비중 더 엄격.
> 4축: ①출제 비중 데이터(근거 있는 문항수) ②풀이 전략 완성도(접근순서·문항당 목표시간·실수포인트·'버릴 문제' 기준) ③대표 예제 카드(구조+풀이단계, ≥8개) ④함정·오답 포인트.
> 표기: O=충분 / △=불충분 / X=미달. 4축 전부 O일 때만 합격. 애매하면 보완.

**언어 소계: 합격 5 / 보완 0.** 비중 합(저빈도 05 포함)이 20문항 내 정합. 독해(01·02) 기본 득점원 + 형식논리/강화약화(03·04) 변별 + 구조(05) 저빈도 고효율로 균형 양호.

## 종합 판정: **합격선 도달 가능** (세 영역 모두 균형 합격)

- **언어 5/5, 자료 5/5, 상황 5/5 — 전 15개 유형 합격. 보완 0.**
- 세 영역 모두 4축(비중 근거·시간/우선순위/포기기준이 박힌 전략·카드 ≥8·함정)을 충족. 합격을 가르는 **'버릴 문제' 기준이 전 유형에서 분 단위 손절 룰(2분 / 2분30초 / 3분 상한)로 구체화**되어 "열심히 풀어라" 수준의 공허한 조언이 한 건도 없음 → 미달 처리 사유 없음.
- **자료해석(시간압박 최악)**도 엄격 기준 적용 결과 합격: 06~10 전부가 어림산·대소비교·30초~3분 손절·전략적 포기(3~4문항 버리고 16~17문항 정확도)를 핵심에 둠. 영역 균형상 가장 우려되던 자료가 오히려 전략적으로 가장 단단함.
- **비중 데이터의 정직성**: 자체 기출 1회(2025)뿐인 한계를 전 섹션이 명시하고, 모(母)시험(국가직 7급·민경채·5급) 분포 환산임을 '추정' 라벨로 표기. 근거를 지어낸 섹션 없음 → '근거 없음 → 보완' 사유 없음.

### 구조적 유의점 (개별 불합격 사유 아님 — 패키지 조립 시 주의)
- **유형별 추정 문항수의 단순 합산이 각 영역 20문항을 초과**한다. 원인: 각 에이전트가 (가) "선지 단위로 편재하는 기반 스킬"(증가율·비율 — 06·07), (나) "퀴즈 전체 vs 순수 퍼즐/계산"(12·13), (다) "광의/협의·부분중첩"(09·10, 11·14, 04·15)을 각각 추정해 **유형 간 중복 카운트**가 발생. 모든 섹션이 이를 '추정·기반스킬·광의'로 라벨링해 정보 왜곡은 아님.
- 대응: `_시간배분표.md`에서 **대표(협의) 추정치로 재배분**해 영역 합계 20에 맞추고, 교시별 총 시간 정합성 코멘트를 달았음. 패키지 조립 시 비중은 시간배분표의 보정치를 정본으로 사용할 것.
