# PSAT 합격 검수 판정표 — 엄격 재채점 **2차** (보강 후 재채점)

> 검수관: 국회직 8급 PSAT **합격 검수관(엄격 재채점 모드 — 2차)**. 직접 조사 없이 `/output/sections/`의 15개 섹션을 **상향된 5축 강화 루브릭**으로 다시 채점.
> **이 표의 위치**: 1차 엄격검수(`판정표-엄격.md`)는 5축 강화 기준으로 **15개 전부 '보완'**(합격 0/보완 15)으로 강등했고, 공통 약점은 ③(worked example 부재)·⑤(블로그·위키·강사 의존)·①(추정 일색)이었다. 그 1차 지적을 받아 각 섹션이 **worked example(실수치)·1차 출처·비중 근거**를 보강한 상태다. 본 2차 표는 그 **보강이 본문에서 실제로 이뤄졌는지**를 직접 검증해, 합격으로 전환됐는지 가린다.

## 채점 방식 (1차 대비)
- 5축: `① 출제 비중 근거 / ② 전략 정량성 / ③ 예제 카드 실질(worked example) / ④ 함정 메커니즘 / ⑤ 출처 질`.
- **5축 모두 O일 때만 '합격'. 하나라도 △/X면 '보완'.** 후하게 주지 않음.
- 1차에서 △였던 축은 **본문에서 보강 실재(worked example이 실제 숫자로 끝까지 계산됐는지 ③, 1차 출처가 실재하는지 ⑤)를 직접 확인**한 뒤에만 O로 승급.
- 표기: O=충분 / △=불충분 / X=미달. 5축 결과는 `비중/전략/카드/함정/출처` 순서.

### 검증 요약 (전 15섹션 공통 — 본문 직접 확인)
- **③ worked example**: 15개 섹션 모두 `▶ worked example` 라벨 + **실수치 계산**이 박혔다(섹션당 5~17개). 자료=교차곱(234×441=103,194 vs 198×512=101,376)·가중평균 박스(80/1,000=8%)·점유율 역산(0.18×2,500=450), 상황=정족수(출석8→과반5, 찬성4=부결)·과태료(10만→5만→4만원)·날짜(6/2 통지→초일불산입→6/16 마감)·가중합(3×8+5×6+2×7=68), 언어=실제 선지↔문장 ○/✗ 대조·벤다이어그램 영역 채움·격자 O/X 전파까지 **끝까지** 계산·판정함.
- **⑤ 출처**: 15개 모두 `## 출처`가 **1차/2차로 분리**됐고, 1차에 **gosi.kr(14)·gosi.assembly.go.kr(12)·공기출 기출원문 0gichul.com/y20xx(14)·국회사무처공고 제2025-22호(11)·공공데이터포털 data.go.kr(8)·KOSIS(2)·법제처 law.go.kr(11번)** 가 실재 링크로 포함됨.
- **①·②·④**: ②(분·초 손절)·④(메커니즘+회피)는 1차에서 이미 거의 O. ①은 회차 실측표(7급 2021~2025 문항번호 단위)·복수 출처 교차·모시험 25→20 환산 근거로 강화됨.

### 1차 → 2차 변화 (요약)

### 합격선 80점 도달 가능성
- **재작성이 아니라 보강으로 합격선에 도달했다.** 1차에서 단단했던 골격(②전략 분·초 손절 + ④함정 메커니즘)에 더해, 2차에서 ③(실전 계산 루틴)·⑤(1차 기출로 훈련 경로 확보)·①(회차 실측 비중)이 채워져 **세 영역 균형 80점(48/60)선 학습 자료로 충분히 작동**한다.
- 특히 시간압박 최악인 **자료해석(06~10)이 카드에 교차곱·가중평균·지수환산·실기출 수치(2025 국회직 8급 매출표·전자책 지수표)를 끝까지 박아** 실전 계산 속도 훈련이 가능해진 점, **상황판단 법조문(11)이 실제 법령(민법·헌법·국회법·질서위반행위규제법) 원문으로 날짜·정족수·과태료를 계산**해 1차 미검증 위험을 제거한 점이 합격 전환의 핵심.
- **종합: 1차 엄격 15보완 → 2차 15합격. 합격선 80점 도달 가능성 충분.** 단, 자체 기출이 2025년 1회뿐이라 ①비중은 모두 "7급 환산 추정" 라벨이 유지됨(루브릭이 25→20 환산을 수치·출처로 제시하면 O로 인정하므로 합격에 지장 없음). 회차가 누적되면 ①을 추정→실측으로 완전 승급 가능.
